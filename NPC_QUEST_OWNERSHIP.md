# NPC Quest Ownership

## Purpose
This document assigns quest ownership so The Last Token does not rely on Mira or Cabinet 07 to carry every story beat.

Each major NPC should own one clear gameplay step, one emotional tone, and one piece of the larger mystery.

## Ownership Rules
- Each required quest needs a named owner.
- The quest owner gives the objective or frames the problem.
- The involved NPC must react after completion.
- Quest completion dialogue should explain why the puzzle mattered.
- Mira remains the emotional anchor, not the universal quest dispenser.
- Cabinet 07 remains an important story machine, not the only machine that advances the game.

## Quest Detail Matrix

### Mira: Lost Token
Location: ArcadeHub / ticket counter

Minigame/puzzle: Rockbyte Duel through Cabinet 07

Story purpose:
- Make the player feel known before they know themselves.
- Establish the token as a missing memory object.
- Show that machines can hold pieces of the player.

Completion reward:
- Lost Token recovered.
- Memory Signal changes from Grounded to Uneasy.
- Mr. Byte / Truth Filter route becomes the next lead.

Completion dialogue direction:
- Mira should say one memory came back.
- She should seem relieved and worried at the same time.
- She should point the player toward machine records, not explain everything.

Foreshadowing line:
> "It remembered you before you did."

Required or optional: Required

### Mr. Byte: Truth Filter
Location: Cabinet Row

Minigame/puzzle: Truth Filter

Story purpose:
- Teach that memories, staff records, and machine statements can disagree.
- Make the mystery feel less emotional and more systemic.
- Move the player from Uneasy to Fractured.

Completion reward:
- Second Memory Fragment recovered.
- Truth Filter cleared.
- Memory Signal changes to Fractured.
- Vendo becomes the next required owner.

Completion dialogue direction:
- Mr. Byte should be precise but incomplete.
- He should acknowledge contradiction as data, not as drama.
- He should suggest signal routing is the next issue.

Foreshadowing line:
> "Record conflict reduced. Identity conflict remains."

Required or optional: Required

### Vendo: Circuit Soda
Location: Snack Alcove

Minigame/puzzle: Circuit Soda

Story purpose:
- Use humor to keep the story from becoming too heavy.
- Translate the player's identity problem into signal flow.
- Make Vendo feel relevant without losing his sarcastic flavor.

Possible puzzle shape:
- Route colored signals through vending-machine channels.
- Match unstable input labels to correct output.
- Use simple visual feedback: stable, misrouted, overflow.

Completion reward:
- Signal Flow stabilized.
- Maintenance Hall route becomes clearer.
- Gus can now address the Staff Door problem.

Completion dialogue direction:
- Vendo jokes about flavor, labels, and return policies.
- Under the joke, he should admit the player's signal is becoming recognizable.

Foreshadowing line:
> "Your label is still missing, but the machine knows what shelf you go on."

Memory Signal effect:
- Remains Fractured, but the hub can become more reactive.

Required or optional: Required

### Gus: Maintenance Sync
Location: Maintenance Hall

Minigame/puzzle: Maintenance Sync / Sync Door

Story purpose:
- Ground the abstract signal story in a practical staff-only problem.
- Make Gus the person who knows how the back rooms actually work.
- Prepare the player for the two-version reveal.

Completion reward:
- Staff Door unlocks.
- Staff Corridor opens.
- Memory Signal becomes Overloaded.

Completion dialogue direction:
- Gus should sound casual about something deeply unsettling.
- He should imply the door listened to two signals, not one.
- He should not reveal Employee 04 fully.

Foreshadowing line:
> "Door heard both knocks. Yours, and the one you forgot making."

Required or optional: Required

### Roxy: Broken High Score
Location: Cabinet Row

Minigame/puzzle: Broken High Score

Story purpose:
- Add optional arcade challenge energy.
- Reinforce corrupted scores and missing player identity.
- Give a future NPC a compact, memorable side role.

Completion reward:
- Optional restored high-score record.
- Optional lore flag.
- Post-reveal changed dialogue.

Completion dialogue direction:
- Roxy should make the restored score feel like a win and a warning.
- She can tease the player while still noticing something is wrong.

Foreshadowing line:
> "Score came back. Name stayed blank. That is usually bad."

Memory Signal effect:
- No required state change.

Required or optional: Optional

### Pip: Prize Counter Secret
Location: Prize Corner

Minigame/puzzle: Prize/trade/observation secret

Story purpose:
- Reward curiosity.
- Give the arcade a softer, stranger corner.
- Tie prizes and tokens to memory without adding inventory complexity.

Completion reward:
- Optional secret completion.
- A small memory detail or post-reveal callback.

Completion dialogue direction:
- Pip should notice tiny details others ignore.
- The secret should feel like a childlike arcade memory turning uncanny.

Foreshadowing line:
> "Someone saved the cheapest prize for the person who never clocked out."

Memory Signal effect:
- No required state change.

Required or optional: Optional

### Cabinet 07: Story Machine
Location: ArcadeHub / Cabinet Row depending on final map layout

Minigame/puzzle: Rockbyte Duel only

Story purpose:
- Guard the Lost Token.
- Establish machine recognition.
- Keep echoing the two-version idea later without owning every quest.

Completion reward:
- Lost Token recovered during Act 1.
- Later interactions provide echoes only.

Completion dialogue direction:
- Cabinet 07 should remain mechanical and unsettling.
- It can acknowledge second fragments or employee signals, but should not become a general quest hub.

Foreshadowing line:
> "Previous profile damaged. Current profile incomplete."

Required or optional: Required for Act 1 only, then supporting role.

## Dialogue Ownership By Phase

### Grounded
- Mira guides.
- Cabinet 07 intimidates.
- Other NPCs offer flavor and objective nudges.

### Uneasy
- Mr. Byte takes ownership.
- Mira reacts emotionally but does not lead every step.
- Machines begin recognizing the player.

### Fractured
- Vendo and Gus become stronger voices.
- Side NPCs can reference contradictions casually.
- Player dialogue should become more specific and unsettled.

### Overloaded
- Gus and Staff Door dominate required guidance.
- NPCs become less able to avoid the truth.
- Machines speak more directly.

### Restored
- All core NPCs can acknowledge Employee 04.
- Optional owners can resolve their smaller mysteries.

## Completion Dialogue Requirement
Every required quest completion must include:
- One owner reaction.
- One player reaction or short first-person observation.
- One clear next objective.

Example structure:
1. Owner acknowledges puzzle result.
2. Player notices a memory or contradiction.
3. Owner points to the next location or NPC.

Keep this short. The game should feel denser, not wordier.
