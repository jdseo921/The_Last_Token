# DESIGN_LOCK.md

## Locked Game Definition
- Title: The Last Token
- Genre: 2D top-down retro arcade mystery
- Setting: Pixel Haven arcade
- Tone: nostalgic, eerie, readable, and grounded

## Core Loop
The player explores the arcade, talks to NPCs, plays small cabinet games, discovers clues, reaches the staff room, sees the identity reveal, and continues in post-reveal roam mode.

## Locked NPC List
- Mira
- Gus
- Vendo
- Cabinet 07
- Mr. Byte

## Locked Minigame List
- Rockbyte Duel
- Cabinet 07 completion sequence

## Locked Twist
The player is Employee 04, a restored memory of the original arcade technician.

## Locked Ending
The MVP ends after the identity reveal, one ending, and save plus continue into post-reveal roam mode.

## Locked Post-Reveal Behavior
- The player can continue exploring after the reveal.
- NPC dialogue changes after the twist.
- The game should remain playable in the same space with updated context.

## MVP Scope
The MVP includes:
- New Memory Slot
- ArcadeHub exploration
- NPC dialogue with the locked NPC list
- Cabinet 07 completion
- Rockbyte Duel
- one simple story puzzle
- staff room access
- slideshow reveal
- one ending
- save and continue
- post-reveal roam mode
- changed dialogue after the twist

## Stretch Scope
Stretch features are allowed only after the MVP loop works end to end:
- Roxy rival NPC
- Pip plush quest
- Broken High Score minigame
- extra cabinet rooms
- multiple endings
- original music
- additional cutscenes
- secret ending
- polished sprite sheets
