# ART_STYLE.md

## Placeholder Art Rules
- Characters should be 32x32.
- Tiles should be 16x16 or 32x32.
- Use a top-down or slight 3/4 RPG perspective.
- Keep a dark arcade palette with readable contrast.
- Favor simple, readable silhouettes.
- Use 2-frame walk cycles only for the MVP.
- Use simple UI panels.
- Slideshow panels can be placeholders first.

## Visual Direction
- Pixel Haven should feel like a dim retro arcade with glowing screens and worn surfaces.
- Important interactables should be easy to read at a glance.
- Mystery and nostalgia matter more than detail density.
- Use minimal animation where it helps clarity.

## UI Rules
- Keep menus and dialogue boxes clean and direct.
- Use placeholder frames if needed, but keep them consistent.
- Do not block core playability waiting for final art.

## Asset Priority
1. Player and core NPC placeholders.
2. ArcadeHub tiles and props.
3. Cabinet interaction art.
4. Slideshow placeholders.
5. Polish only after the MVP loop works.
